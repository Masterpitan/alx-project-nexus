POST Create Post

http://localhost:8000/api/posts/create/

Create a new post

Body: {"user_id": 1, "content": "Post content"}

POST Create Comment

http://localhost:8000/api/comments/create/

Add a comment to a post

Body: {"post_id": 1, "user_id": 1, "text": "Comment text"}

POST Create Interaction

http://localhost:8000/api/interactions/create/

Like or dislike a post

Body: {"post_id": 1, "user_id": 1, "reaction": "like"}
